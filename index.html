Today we will discover how computers make decisions using IF and ELSE statements.

Just like humans make choices every day, computers also make choices by checking conditions.

What is a Condition?

A condition is a question that has only two possible answers:

✓ TRUE (Yes)
✗ FALSE (No)

A computer checks the condition and follows the correct path — just like you decide what to do based on what you see around you!

Flowchart

Pseudocode
Pseudocode
START

⬇
TRUE ✓

FALSE ✗

📑 Key Vocabulary

IFChecks a condition. If it is TRUE, this action runs.
ELSEIf the condition is FALSE, this action runs instead.
ConditionA question that is either TRUE or FALSE.
